# Madison CV Website

## Files

- `index.html` — page content and links
- `styles.css` — complete layout and responsive styling
- `script.js` — small navigation scroll helper
- `assets/images/` — place all website images here

## Required image filenames

Add these files inside `assets/images/`:

- `profile-photo.png`
- `project-one.jpg`
- `project-two.jpg`
- `project-three.jpg`

A transparent PNG is recommended for the profile photo.

## Change your links

Open `index.html` and replace:

- `YOUR_GITHUB_URL`
- `YOUR_LINKEDIN_URL`
- `YOUR_PROJECTS_URL`

Also update the phone number, email addresses, physical address and certification text.

## Run the site locally

You can double-click `index.html`, but using the VS Code **Live Server**
extension is better.

1. Open the folder in VS Code.
2. Install the Live Server extension.
3. Right-click `index.html`.
4. Select **Open with Live Server**.

## Publish later

This static website can be hosted on:

- GitHub Pages
- Netlify
- Vercel
- Cloudflare Pages

## Included image setup
- `OnlineCVBackground.png` is applied automatically to every `.panel` section through CSS at low opacity.
- `profilePic.png` is used as the hero headshot.
- The three supplied WhatsApp images were copied to `project-one.jpg`, `project-two.jpg`, and `project-three.jpg` so all project previews load.
